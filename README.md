\# 🔮 Die Börsen-Glaskugel (Community Edition)



Ein transparentes Tool für hochspekulative Marktdaten im März 2026. 



\### Ziele des Projekts:

\- \*\*Transparenz:\*\* Offenlegung von Short-Quoten und Insider-Daten.

\- \*\*Bildung:\*\* Verständnis von Risiko-Management (Stop-Loss).
